# Security policy

## Reporting a vulnerability

Please do not publish sensitive vulnerabilities, credentials, personal data, or working exploitation details in a public issue.

Use GitHub's private vulnerability reporting feature when it is available for this repository. Include the affected version, reproduction steps, expected impact, and a suggested fix if you have one.

## Scope

Cros is a local-first research and defensive-security workspace. Reports involving unauthorized access, leaked data sales, credential abuse, doxxing, or attacks against systems without permission are outside the project's intended use.
